# Smart Warehouse Robotics Decision Engine

Graduation project that simulates a smart warehouse robot decision system using Java Spring Boot, SQL Server, Thymeleaf, Spring Security, Interpreter Pattern, and Strategy Pattern.

## Features
- Role-based login for Admin, Manager, and Staff
- Robot, rule, user, and mission management
- Live Map robot movement simulation
- Robot Task Board
- Rule Engine using Interpreter Pattern
- Strategy Engine using Strategy Pattern
- Battery, charging, obstacle, priority, and load handling

## Tech Stack
Java, Spring Boot, Spring Security, Thymeleaf, SQL Server, Maven, HTML, CSS, JavaScript, Bootstrap

## Notes
This is an academic/demo project.
